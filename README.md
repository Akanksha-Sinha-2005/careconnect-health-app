# 💙 CareConnect – Healthcare Support Web App

## 📌 Overview
CareConnect is a mini healthcare support web application that allows patients and volunteers to submit requests and receive instant AI-based responses.

## 🚀 Features
- User-friendly healthcare support form
- Patient & Volunteer registration
- SQLite database integration
- AI-based auto-response chatbot
- Clean and modern UI (Bootstrap)

## 🤖 AI Feature
The system includes a rule-based chatbot that:
- Detects keywords in user queries
- Provides instant automated responses
- Reduces manual workload

## 🛠 Tech Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: Python (Flask)
- Database: SQLite
- Hosting: Render

## 🌍 NGO Use Case
This app can help NGOs:
- Collect patient requests efficiently
- Provide instant replies to common queries
- Track and store user data
- Improve response time in emergencies
